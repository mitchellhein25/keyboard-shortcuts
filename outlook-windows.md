### Navigation & View
| Shortcut | Action |
|---------|--------|
| Ctrl + 1 | Switch to Mail |
| Ctrl + 2 | Switch to Calendar |
| Ctrl + E | Search |
| Ctrl + Y | Go to Folder Pane |
| Alt, H | Move focus to Home tab |
| Alt, V | Move focus to View tab |
| ← | Collapse section |
| → | Expand section |

### Message Actions
| Shortcut | Action |
|---------|--------|
| Ctrl + R | Reply |
| Ctrl + Shift + R | Reply All |
| Ctrl + Enter | Send |
| Shift + Enter | Open message in new window |
| Ctrl + Shift + V | Move to folder |

### Read / Status Management
| Shortcut | Action |
|---------|--------|
| Ctrl + Q | Mark as read |
| Ctrl + U | Mark as unread |
| Insert | Flag message |

### Deletion
| Shortcut | Action |
|---------|--------|
| Shift + Delete | Permanently delete |
