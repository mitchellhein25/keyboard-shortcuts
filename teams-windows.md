## Microsoft Teams (Windows)

### Global Navigation
| Shortcut | Action |
|---------|--------|
| Ctrl + E | Global search |
| Ctrl + G | Open existing chat or channel |
| Ctrl + 1 | Open 1st app |
| Alt + ← | Go back |
| Alt + → | Go forward |
| Ctrl + F6 | Go to next section |
| Ctrl + Shift + F6 | Go to previous section |

### Focus & Pane Management
| Shortcut | Action |
|---------|--------|
| Ctrl + L | Focus Chat / Channel list |
| Ctrl + M | Focus message pane |
| Ctrl + R | Focus compose box |
| Ctrl + Alt + Enter | Move focus to pane divider |
| Ctrl + Shift + Enter | Reset pane to default width |
| Ctrl + Alt + T | Focus top toast notification |

### Chats & Channels
| Shortcut | Action |
|---------|--------|
| Ctrl + N | New chat |
| Ctrl + Shift + N | New chat in new window |
| Ctrl + O | Pop out existing chat or channel |

### Messaging & Composition
| Shortcut | Action |
|---------|--------|
| Ctrl + Enter | Send message |
| Shift + Enter | New line |
| Ctrl + Shift + X | Expand compose box |
| Ctrl + K | Insert link |
| Alt + Shift + O | Attach file |
| Ctrl + F | Search current chat or channel |

### Replies & Reactions
| Shortcut | Action |
|---------|--------|
| Alt + Shift + R | Reply to latest message |
| Ctrl + Alt + R | React to latest message |
